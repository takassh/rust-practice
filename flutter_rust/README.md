# What is this?
- Flutter with Rust project using https://github.com/fzyzcjy/flutter_rust_bridge?tab=readme-ov-file

# How can I set up?
- just run this commands
```
cargo install 'flutter_rust_bridge_codegen@^2.0.0-dev.0' && \
    flutter_rust_bridge_codegen create my_app && cd my_app && flutter run
```
